# Claude Code 设计思想与核心架构

> 深入理解 AI 辅助开发工具的设计哲学与实现原理

## 📖 在线阅读

**https://humorousq.github.io/claude-code-design-book/**

## 📚 内容简介

本书深入探讨 Claude Code 的设计思想、核心架构和实现原理，帮助开发者理解 AI 辅助开发工具的内在逻辑。

### 核心内容

- **设计哲学** - AI First 的设计理念
- **Agent 系统** - 多 Agent 架构与协作机制
- **Memory 系统** - 持久化知识存储与检索
- **工具系统** - 文件操作、搜索、执行等核心工具
- **MCP 集成** - Model Context Protocol 的深度应用
- **提示工程** - System Prompts 的设计艺术
- **性能优化** - Token 管理与缓存策略

### 目录结构

- 前言
- 第 1-14 章：核心架构与技术实现
- 附录 A-E：命令参考、工具参考、最佳实践等

## 🚀 本地开发

```bash
# 安装依赖

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build

# 预览构建结果
npm run docs:preview
```
